# Задача 1: Сумма чисел
#### Попросите пользователя ввести положительное целое число. Затем используйте цикл for или while для вычисления суммы всех чисел от 1 до введенного числа.
### Пример вывода:
```bash
Введите число: 4
Сумма: 9
```
